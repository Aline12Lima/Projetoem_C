## Resumo ##
O presente trabalho tem como objetivo desenvolver um sistema de gerenciamento
para salões de beleza, utilizando a linguagem de programação C. O sistema, denominado
SalonSystem, permitirá realizar operações de cadastro, consulta, atualização e exclusão
(CRUD) de clientes, serviços e agendamentos. A aplicação buscará otimizar o controle de
horários e registros, substituindo métodos manuais e reduzindo erros em marcações. Além
das operações básicas, será implementada uma funcionalidade adicional para gerar
relatórios mensais, contendo informações sobre os serviços realizados, clientes atendidos e
o total arrecadado no período. O projeto visa aplicar conceitos de programação estruturada,
como o uso de structs, modularização, manipulação de arquivos e funções, contribuindo
para a organização e eficiência administrativa de salões de beleza.
